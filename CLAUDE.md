## Project

This is the source for choromusi.ca — a choro music website deployed via GitHub Pages.

## Rules

- Always commit and push changes when done with a task so results are visible on the live site.
- Always merge changes to `main` when done so the site deploys and results are visible.
- The site deploys automatically to GitHub Pages on every push to `main`.
- The GitHub Pages URL is: https://slindbergh.github.io/choromusi.ca/
